## 보건/의료 시설 맞춤형 시각화 서비스
병원, 응급실, 비급여와 같은 다양한 보건/의료 정보를 사용자 맞춤형으로 제공하고, 데이터 분석을 통해 실용성있는 데이터 시각화를 제공한다.
